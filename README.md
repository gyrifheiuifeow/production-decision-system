# 生产决策优化系统

> 基于数学建模的企业生产过程决策优化方案

## 功能特点
- 序贯概率比检验(SPRT)抽样方案
- 多阶段成本效益模型
- 遗传算法/模拟退火优化
- 可视化决策分析

## 技术栈
- 前端：HTML/CSS/JavaScript + Chart.js
- 后端：Python Flask
- 核心算法：SPRT/GA/SA

## 安装运行
```bash
# 克隆项目
git clone git@github.com:gyrifheiuifeow/production-decision-system.git

# 进入项目目录
cd production-decision-system

# 安装依赖
pip install -r requirements.txt

# 启动系统
python app.py
